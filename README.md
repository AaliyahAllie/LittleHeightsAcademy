Little Heights Academy – User Support Manual
1. Overview
The Little Heights Academy application is designed to simplify student registration and manage school activities for children aged 1–5. The system provides an efficient, paperless way for parents to register students, track fees, view reports, and for school staff to manage students, invoices, and seating arrangements.
The app also supports extracurricular activities such as Islamic teachings, swimming, karate, and junior soccer, allowing staff to keep track of students’ participation.
2. System Features
The app is divided into several functional areas for different types of users: Admins, Staff, and Parents.
2.1 Admin Features
•	Add, edit, and delete student records.
•	Manage parent accounts.
•	Track invoices for registration or extracurricular activities.
•	Assign seating arrangements for classes.
•	View student reports and activity participation.
2.2 Parent Features
•	Register students online.
•	View student details (name, class, activities).
•	View invoices and payment status.
•	Receive notifications regarding school events or payments.
2.3 General Features
•	Secure login for all users.
•	Dashboard displaying key information.
•	Manage extracurricular activities.
•	Generate reports for student performance.
3. User Access
•	Admin Users: Full access to all system features, including student, parent, invoice, and report management.
•	Parents: Limited access to their child’s records, invoices, and activity participation.
•	Staff/Teachers: Access to student attendance, class reports, and activity management (optional based on setup).
4. Getting Started
4.1 Logging In
1.	Open the Little Heights Academy application.
2.	Enter your username and password.
3.	Click Login.
4.	Admin users will see the full dashboard; parents will see their child’s information and invoices.
4.2 Navigation
•	Dashboard: Displays overview of students, invoices, and activities.
•	Students: View and manage student information.
•	Parents: Manage parent accounts and contact details.
•	Invoices: View invoice history, payment status, and create new invoices (Admins only).
•	Reports: View student performance in classes and extracurricular activities.
•	Seating: Track class seating capacity and availability.
5. Managing Students
5.1 Adding a Student (Admin)
1.	Go to the Students section.
2.	Click Add Student.
3.	Fill in:
o	First Name, Last Name
o	Date of Birth
o	Address
o	Email
o	Activities (e.g., Swimming, Karate)
4.	Click Save to register the student in the system.
5.2 Viewing Student Details
•	Click on a student’s name to see:
o	Personal information
o	Activities
o	Performance reports
o	Parent information
6. Managing Parents
•	Admin users can add or update parent accounts.
•	Parents can log in to view their child’s details.
•	Contact information is stored securely in the system.
7. Invoices
•	Admin Users can create invoices for student registration or extracurricular activities.
•	Parents can view invoices and check payment status.
•	Invoices include:
o	Amount
o	Description
o	Date
o	Status (Pending/Paid)
8. Reports
•	View class performance and activity marks.
•	Reports are timestamped and stored per student.
•	Admins can generate reports for monitoring overall student progress.
9. Seating
•	Shows class seating availability and capacity.
•	Helps allocate students efficiently based on class size.
•	Updated automatically when students are registered or removed
10. Security and Privacy
•	All user accounts require login credentials.
•	Parents only see their child’s data.
•	Admins have full access but are responsible for safeguarding sensitive data.
•	Data is stored securely in the Firebase database.
11. Troubleshooting
Problem	Solution
Cannot log in	Ensure correct username and password; reset password if necessary.
Cannot see invoices	Check if your account is linked to a student; contact admin if missing.
Student info not updating	Refresh the page; ensure admin has saved changes.
App crashes	Restart the app; contact support if the issue persists.
